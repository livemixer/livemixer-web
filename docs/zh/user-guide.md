# 用户使用手册

本指南介绍如何使用 LiveMixer Web Studio 进行实时视频场景编辑和推流。

## 界面布局

主界面由以下区域组成：

```
+----------------------------------------------------------+
|  Logo  |         工具栏（菜单）             | 用户区域   |
+--------+-----------------------------------+-------------+
|         |                                   |             |
| 左侧    |       画布（Konva.js）            |   右侧      |
| 边栏    |       可视化场景编辑器             |  边栏       |
| （拉流） |                                   | （属性面板） |
|         |                                   |             |
+--------+-----------------------------------+-------------+
|  场景   |   源列表   |  音频混音  | 推流控制              |
+----------------------------------------------------------+
|  状态栏                                                  |
+----------------------------------------------------------+
```

- **工具栏** - 包含 File、Edit、View、Tools、Help 菜单
- **左侧边栏** - 参会者面板和拉流控制
- **画布** - 基于 Konva.js 的主编辑区域
- **右侧边栏** - 选中源的属性面板
- **底部栏** - 场景管理、源列表、音频混音和推流控制
- **状态栏** - 显示推流状态、分辨率、FPS 和 CPU 使用率

## 场景管理

场景是画布上排列的源的集合。你可以创建多个场景，在直播过程中切换。

### 创建场景

在场景面板（左下角）点击 **+** 按钮添加新场景。场景按序号命名（Scene 1、Scene 2 等）。

### 切换场景

在场景面板中点击场景标签即可切换，画布将更新为所选场景的源。

### 排序场景

使用场景旁边的上下箭头按钮调整显示顺序。

### 删除场景

点击场景旁边的删除按钮。必须至少保留一个场景。

## 源类型

源是放置在画布上的视觉和音频元素。可用的源类型如下：

| 源类型 | 插件 | 说明 |
|--------|------|------|
| `video_input` | 摄像头 | 从摄像头设备采集视频 |
| `audio_input` | 音频输入 | 从麦克风采集音频，带混音控制 |
| `screen` | 屏幕捕获 | 捕获整个屏幕 |
| `window` | 屏幕捕获 | 捕获特定应用窗口 |
| `mediasource` | 媒体源 | 播放 URL 视频 |
| `image` | 图片 | 显示图片叠加 |
| `text` | 文字 | 渲染文字叠加 |
| `timer` | - | 倒计时或正计时 |
| `clock` | - | 实时时钟 |
| `livekit_stream` | - | 来自 LiveKit 的远端参会者流 |

### 添加源

1. 在源面板（底部中间）点击 **+** 按钮
2. 从对话框中选择源类型
3. 根据源类型：
   - **摄像头/音频**：先显示设备选择对话框
   - **屏幕捕获**：立即弹出浏览器权限提示
   - **媒体源**：先添加，再在属性面板配置 URL
   - **图片/文字**：直接添加，在属性面板配置
   - **计时器/时钟**：弹出配置对话框

### 删除源

在列表中选中源后点击删除按钮，或在画布上选中后按 `Delete` 键。

## 画布操作

画布是核心编辑区域，用于可视化排列你的源。

### 选择

点击画布上的源进行选择，选中的源会显示变换手柄。

### 移动

点击并拖动源来重新定位。

### 缩放

拖动选中源的边角或边缘手柄来调整大小。按住 `Shift` 保持纵横比。

### 图层顺序

使用源面板中的上下箭头按钮更改 z-order。列表中靠上的源渲染在上层。

### 锁定

点击源旁边的锁定图标防止误操作。锁定的源在画布上显示锁定指示器。

### 可见性

点击源旁边的眼睛图标切换可见性。隐藏的源仍存在于场景中，但不会在画布上渲染或包含在推流中。

## 属性面板

右侧边栏显示当前选中源的属性面板。可用属性因源类型而异：

- **布局** - 位置（X, Y）和尺寸（宽度, 高度）
- **变换** - 不透明度、旋转、圆角
- **源特有** - 设备选择、URL、文本内容、字体大小、颜色等
- **音频** - 音量滑块和静音切换（支持音频的源）

属性面板中的更改会立即应用到场景。

## 音频混音

底部栏的音频混音面板允许你控制支持音频的源的音频级别：

- **音量滑块** - 调整每个音频源的音量
- **静音切换** - 快速静音/取消静音

支持音频的源（如 `audio_input`、`video_input`、`mediasource`）会自动出现在音频混音器中。

## 参会者面板

左侧边栏包含参会者面板，连接到 LiveKit 房间拉流时显示远端参会者。

### 拉流

1. 在 **Settings > Pull** 中配置 LiveKit 拉流 URL 和 Token
2. 点击左侧边栏的 **Connect** 按钮
3. 远端参会者将出现在列表中

### 添加参会者到场景

点击参会者旁边的摄像头或屏幕共享图标，将其流作为 `livekit_stream` 源添加到当前场景。

## 设置对话框

通过齿轮图标或 **File > Settings** 打开设置对话框。包含以下选项卡：

### 视频设置

- **基础分辨率** - 画布基础分辨率（默认：1920x1080）
- **输出分辨率** - 输出流分辨率
- **FPS** - 目标帧率（默认：30）
- **缩放滤镜** - 缩放算法（bilinear）

### 音频设置

- **音频设备** - 默认音频设备
- **采样率** - 音频采样率（默认：48000）
- **声道** - 音频声道配置（stereo/mono）

### 推流设置

- **推流服务** - 推流服务类型
- **LiveKit URL** - LiveKit 服务器 URL
- **LiveKit Token** - 访问令牌（出于安全不持久化）
- **视频码率** - 视频码率，单位 kbps（默认：5000）
- **视频编码器** - 视频编码（vp8, h264, vp9, av1, h265）

### 拉流设置

- **LiveKit 拉流 URL** - 用于拉流的 LiveKit 服务器 URL
- **LiveKit 拉流 Token** - 拉流访问令牌（不持久化）

### 视图设置

- **显示网格** - 在画布上显示网格覆盖
- **显示参考线** - 移动源时显示对齐参考线

### 场景转场

- **转场类型** - 场景间切换动画（cut, fade, dissolve, swipe, stinger）
- **转场时长** - 持续时间，单位毫秒（默认：300）

## 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` 或 `Ctrl+Shift+Z` | 重做 |
| `Ctrl+C` | 复制选中源 |
| `Ctrl+V` | 粘贴复制的源 |
| `Delete` / `Backspace` | 删除选中源 |

> 文本输入框获得焦点时，快捷键将被禁用。

## 配置导入导出

场景配置可以 JSON 文件导入和导出：

- **导出** - 将当前场景布局保存为 JSON 文件
- **导入** - 从 JSON 文件加载之前保存的场景布局

JSON 格式遵循 [Protocol v1.0.0 规范](./protocol.md)。
